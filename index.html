What is this?

Welcome to Page Ideas by Sergie, a tool desgined to explore different ideas for web development.

Here you will be able to find web pages designed for different types of businesses and with a multitude of web technologies so you can get inspired and hopefully learn a few neat tricks.

This is an ongoing project being constantly updated. Come back regularly for new exciting features and the latest knowledge on web development!

Upcoming features: Responsivity, Filters, Preview, Extra resources, Customized experience, animations and more!
